Deep learning project for classifying brain tumors from MRI images. Uses transfer learning with ResNet50 fine-tuned on a medical imaging dataset. Implements data augmentation, class balancing, and Grad-CAM visualizations for model interpretability. Built with TensorFlow/Keras and includes a Streamlit demo interface for uploading and classifying MRI scans.
