Custom-designed multi-sensor module built around the ESP32 microcontroller. Integrates MPU6050 (accelerometer/gyro), MAX6675 (thermocouple), and DHT22 (temperature/humidity) sensors. Publishes data over MQTT with configurable intervals, OTA firmware updates, and WiFi reconnection handling. Deployed in FabCity Hamburg FabLabs as part of the LAUDS project.
